First row: Keyboard/editor mode
Arrow keys = dpad
Enter = A
Backspace = Y
LR = Left/Right shift
Select = Escape

Second row: Button/game mode
WASD = dpad
Arrow keys = ABXY
LR = Left/Right shift
Enter = Start
Select = Escape

See https://www.sfml-dev.org/documentation/2.6.1/classsf_1_1Keyboard.php
to determine key codes.

Order: Up Down Left Right A B X Y L R START SELECT [MODE]
MODE switches between control scheme 1 and 2.
